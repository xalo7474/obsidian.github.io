Getting Started

Follow these instructions to set up Obsidian in your project.

Installation

npm install Obsidian-framework

Import the CSS file in your HTML:

<link rel="stylesheet" href="path/to/Obsidian.css">

Ensure you have included the Font Awesome library for icons:

<link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/font-awesome/5.15.4/css/all.min.css">

Advanced Features

Explore some advanced functionalities you can use with Obsidian.

Custom Scrollbars

Style your scrollbars with these CSS rules:


/* Custom Scrollbar */
::-webkit-scrollbar {
    width: 8px;
}
::-webkit-scrollbar-track {
    background: #1E1E1E;
}
::-webkit-scrollbar-thumb {
    background: var(--accent-color);
    border-radius: 10px;
}
::-webkit-scrollbar-thumb:hover {
    background: #61428b; /* Darken on hover */
}

Neumorphism Effect

To create a neumorphic effect, use the following class:

.neumorphic {
    background: #1E1E1E;
    border-radius: 20px;
    padding: 20px;
    box-shadow: 8px 8px 20px #0A0A0A, -8px -8px 20px #292929;
}
